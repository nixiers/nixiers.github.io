# NIXIERS — Developer Portfolio

Темное футуристическое портфолио для GitHub Pages: статический сайт на HTML, CSS и JavaScript без сборки.

## Локальный запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

После запуска сайт будет доступен по адресу `http://localhost:8000`.

## Как менять текст

Основной контент находится в `index.html`: hero, блок "Обо мне", навыки, проекты, услуги и контакты. Меняйте тексты прямо в соответствующих секциях.

## Как менять проекты

Проекты находятся в секции `#projects` файла `index.html`. Для каждого проекта есть карточка `.project-card` с названием, описанием и тегами.

## Как менять соцсети

Ссылки на GitHub, Discord, Telegram и Email находятся в блоках `.socials` и `.contact-list` в `index.html`. Замените `href` и видимый текст на актуальные контакты.

## Как менять дизайн

Весь стиль сайта находится в `style.css`. Основные цвета вынесены в CSS-переменные в начале файла: `--bg`, `--purple`, `--violet`, `--blue`, `--text`, `--muted`.

## Деплой на GitHub Pages

Репозиторий `nixiers.github.io` публикуется GitHub Pages из корня. Для деплоя достаточно закоммитить и отправить файлы в основную ветку:

```bash
git add .
git commit -m "Redesign portfolio"
git push
```

Сайт будет открываться по адресу `https://nixiers.github.io`.
